# Network Security Lab

This repository contains practical documentation of enterprise network security concepts including firewall configuration, VPN setup, routing, and secure network architecture.

It is based on real-world exposure to enterprise environments using Sophos, Fortinet, and standard networking infrastructure.

---

## 🌐 Focus Areas

- Firewall Configuration & Policy Management
- VPN Setup (Site-to-Site / Remote Access)
- NAT & Port Forwarding
- Network Troubleshooting
- Access Control & Security Policies
- Secure Network Design Concepts

---

## 🔐 Firewall Technologies

- Sophos Firewall
- Fortinet Firewall
- Palo Alto (Exposure)

---

## 🌍 Networking Concepts Covered

- TCP/IP Model
- DNS & DHCP
- Routing & Switching Basics
- NAT (Network Address Translation)
- LAN / WAN Architecture
