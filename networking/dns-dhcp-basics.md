# DNS & DHCP Basics

## 🎯 Objective
To manage network name resolution (DNS) and dynamic IP allocation (DHCP) in enterprise environments.

---

## 🌐 DNS (Domain Name System)

- Converts domain names to IP addresses
- Example: google.com → 142.x.x.x

### Key Functions:
- Name resolution
- Load distribution
- Domain mapping

---

## 🧠 DHCP (Dynamic Host Configuration Protocol)

- Automatically assigns IP addresses to devices
- Reduces manual IP configuration

### DHCP Process (DORA):
1. Discover
2. Offer
3. Request
4. Acknowledge

---

## 🛡 Security Aspects

- Prevent rogue DHCP servers
- Monitor DNS queries for malicious domains
- Use DNS filtering for threat protection
