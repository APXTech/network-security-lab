# Routing Basics

## 🎯 Objective
To understand how data is routed between different networks using routing protocols and tables.

---

## 🌐 What is Routing?

Routing is the process of selecting the best path for data packets to travel across networks.

---

## 🔄 Types of Routing

### 1. Static Routing
- Manually configured routes
- Simple but not scalable

### 2. Dynamic Routing
- Automatically updates routes
- Uses protocols like OSPF, RIP

---

## 📊 Routing Table Contains:

- Destination IP
- Subnet Mask
- Gateway
- Interface

---

## 🛡 Security Perspective

- Prevent route hijacking
- Secure routing updates
- Monitor unusual traffic paths
